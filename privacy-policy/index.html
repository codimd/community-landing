Privacy Policy

Preamble: Hosting Details

The HedgeDoc landing page is a static site that does not track, collect, store any information, or set any cookies. There is no account on the website itself.

There are some third party places where the HedgeDoc Project stores data:

  • GitHub Pages is the website host. Please refer to GitHub’s privacy practices document for details.
  • Our Chat is located on matrix.org at #hedgedoc:matrix.org. Refer to matrix.org’s privacy policy for details.
  • The demo instance stores login information (email addresses, password, GitHub Access Tokens, Twitter Access Tokens) and note information (content, revision and so on). For details refer to the privacy notice of the demo instance.
  • The HedgeDoc 2 Frontend UI Test is hosted by Netlify. Please refer to Netlify’s privacy policy for details.
  • We thankfully use POEditor for all our translations. Refer to POEditor’s privacy practices document for details.
  • The HedgeDoc code is stored in git, on GitHub. The git software does collect names and email addresses in repositories. GitHub has account information. Refer to GitHub’s privacy policy for details.
